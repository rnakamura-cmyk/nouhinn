# クラウドワークス 応募手順

## サイト情報
- URL: https://crowdworks.jp
- ログイン: ブラウザで事前にログイン済みであること

## 検索手順
1. https://crowdworks.jp にアクセス
2. カテゴリ・報酬・募集中フィルタで検索（config.json の conditions を参照）
3. 検索結果から10件程度を確認する
4. **このステップでは応募しない。PIPELINE_SKILL.md の Step 4 でユーザー確認を取ってから応募する。**

## 応募フォームの埋め方（Step 5 で使用）

| フィールド | 対応 |
|-----------|------|
| 契約金額の提示 | 「相談してから金額を提案」を選択 |
| 契約金額（税抜） | 入力不要 |
| 完了予定日 | 空欄のまま |
| 応募有効期間 | 「設定しない」 |
| メッセージ | 案件専用の応募文を入力 |
| 添付ファイル | なし |

## 出力ファイル名
`crowdworks_YYYYMMDD_HHMMSS.jsonl`
