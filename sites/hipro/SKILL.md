# HiPro Direct 応募手順

## サイト情報
- URL: https://talent.direct.hipro-job.jp/talent/for-local/top/
- ログイン: ブラウザで事前にログイン済みであること
- 特徴: 大手企業・地方企業の副業・兼業プロジェクト。ハイクラス人材向け

---

## 検索手順

`config.json` の `search_filters` を参照して以下の手順で操作する。

### 1. 案件一覧ページにアクセス

> ✅ 確認済み（2026-04-01）：検索URL構造を実機で確認済み。

**検索ベースURL**: `https://talent.direct.hipro-job.jp/talent/issue/new/`

ログイン後、ナビバーの「案件検索」から遷移可能。

### 2. 職種カテゴリで絞り込み

URLパラメータで制御可能：

| パラメータ | 説明 | 値 |
|-----------|------|-----|
| `themeType` | 職種タブ | 1=ビジネス職, 2=IT・クリエイティブ職 |
| `freeWords[0]` | キーワード（複数可） | 自動化, DX, AI等 |
| `sortType` | 並び順 | 1=おすすめ順 |
| `isIncludeForLocal` | for Local含む | true/false |
| `page` | ページ | 1〜 |
| `perPage` | 1ページ件数 | 30 |

**注意**: URLの`keyword`パラメータは機能しない。必ずUI上のキーワード検索ボックスに入力して「検索」ボタンをクリックするか、`freeWords[0]`パラメータを使用すること。

サイドバーフィルター（UIで操作）:
- カテゴリ（ドロップダウン）
- 都道府県（ドロップダウン）
- 案件形式: プロジェクト / スポットコンサル
- 契約形態: 直請け型 / 再委託型
- その他: 募集終了案件を含む / for Local案件を含む

### 3. キーワード検索
検索ボックスに `preferred_keywords` のキーワードを入力して「検索」ボタンをクリック。
「自動化」キーワードがプロフィールとの合致度が高い（77件ヒット, 2026-04-01時点）。

### 4. 案件を収集
- 表示された案件を上から順に確認（最大20件程度）
- PIPELINE_SKILL.md Step 3 の基準でフィルタリング
- `preferred_keywords` に合致する案件を優先
- HiPro Directはハイクラス人材向けのため、報酬5,000円/h以上の案件が中心。月120h以上（=フルタイム相当）の案件は副業向きではないためスキップ推奨

## 応募フォームの埋め方

> ✅ 確認済み（2026-04-01）：応募フォームはモーダル形式。案件詳細ページの「応募に進む」ボタンで開く。

| フィールド | 種類 | 入力内容 |
|-----------|------|--------|
| 応募コメント | テキストエリア（800文字制限） | PIPELINE_SKILL.md Step 4 のルールで生成。ヒント：「惹かれた理由」「参加した際にできること」を含める |
| 稼働条件チェック① | チェックボックス | 報酬金額の目安（案件ごとに表示が変わる）→ 合意できればチェック |
| 稼働条件チェック② | チェックボックス | 稼働時間の目安（案件ごとに表示が変わる）→ 合意できればチェック |
| 稼働条件チェック③ | チェックボックス | 活動場所（案件ごとに表示が変わる）→ 合意できればチェック |

**送信フロー:**
1. 「応募に進む」ボタン → モーダル表示
2. 応募コメント入力 + 稼働条件チェック
3. 「応募内容を確認」ボタン → 確認画面
4. 確認画面で「応募する」ボタン → 送信完了

**注意:**
- フォームはReactベース → setNativeValueパターンでテキストエリアに値を入力すること
- チェックボックスは未チェックでも「相談希望」として送信可能
- 添付ファイル欄は無し（プロフィールが自動送信される模様）
- モーダルを閉じると入力内容は破棄される
