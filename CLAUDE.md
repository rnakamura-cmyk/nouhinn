# 案件応募自動化システム

このシステムは、副業サイトで案件を自動検索・応募し、結果をGoogleスプレッドシートに記録するツールです。

```

y で進める / n でキャンセル
```

### 1-1. システムパスの自動検出（OS対応）

以下のPythonスクリプトを実行して、このシステムの絶対パスを `config.json` の `base_dir` に書き込む：

```bash
python -c "
import json, pathlib, sys
base = str(pathlib.Path('.').resolve())
p = pathlib.Path('config.json')
cfg = json.loads(p.read_text(encoding='utf-8'))
cfg['base_dir'] = base
p.write_text(json.dumps(cfg, ensure_ascii=False, indent=2), encoding='utf-8')
print('OS:', sys.platform)
print('base_dir:', base)
"
```

実行後、表示されたパスをユーザーに確認する：

```
パスを設定しました：
  OS: {表示されたOS}
  フォルダ: {表示されたパス}

このパスで合っていますか？（y / 違う場合は正しいパスを入力）
```

「y」以外が返ってきた場合は、入力されたパスで `config.json` の `base_dir` を手動で更新してから次へ進む。

## フェーズ2：プロフィール設定

以下の質問を1つずつ聞いてください。まとめて聞かず、返答を受けてから次の質問へ。

**Q1**
```
あなたのお名前・ビジネスネームを教えてください。
（例：むらりょう / 中村崚）
```

**Q2**
```
一言で表すと、どんな仕事が得意ですか？
（例：GASとAIで業務を自動化するフリーランス）
```

**Q3**
```
得意な領域・スキルを教えてください。箇条書きでOKです。
（例：GAS・スプレッドシート自動化、n8n/Difyでのワークフロー構築、など）
```

**Q4**
```
実績を1〜3つ教えてください。数字があると◎です。
（例：受注管理を自動化して月20時間の削減を実現、など）
```

**Q5**
```
応募したくない仕事の条件はありますか？
（例：常駐・対面必須・デザイン系・語学系、など）
```

**Q6**
```
希望の単価・稼働時間はありますか？（任意）
（例：月5万円〜、週5〜10時間、など）
```

全回答を受け取ったら `my_profile.md` に以下の形式で書き出す：

```markdown
# プロフィール

## 基本情報
- 名前: {Q1の回答}
- キャッチコピー: {Q2の回答}

## 得意領域
{Q3の回答を箇条書きで}

## 実績ハイライト
{Q4の回答を箇条書きで}

## 応募NG条件
{Q5の回答を箇条書きで}

## 希望条件
{Q6の回答}
```

書き出し後、「プロフィールを保存しました！」と伝える。

## 日常の使い方

### Coworkへの指示（案件応募）

```
# クラウドワークスで応募
案件応募を実行してください。クラウドワークスで、[カテゴリ名]で。

# スキルシフトで応募
案件応募を実行してください。スキルシフトで、[カテゴリ名]で。

# 複数サイト同時
案件応募を実行してください。クラウドワークスとスキルシフト両方で。
```

### スプレッドシートへの同期

```bash
python3 sync_sheets.py
```

または5分ごとに自動実行されるスケジュールタスク「crowdworks-auto-sync」が自動処理します。

## ファイル構成

```
案件自動応募/                  ← フォルダ名はインストール先によって異なる
├── CLAUDE.md              ← このファイル（セットアップ案内）
├── COWORK_SKILL.md        ← Coworkへの指示ルーター
├── my_profile.md          ← あなたの強み・実績（セットアップで自動生成）
├── my_profile.example.md  ← プロフィールの記入例
├── config.json            ← base_dir・スプレッドシートID・カラム定義
├── cowork_config.json     ← 検索条件・応募基準設定
├── sync_sheets.py         ← Google Sheets同期スクリプト
├── requirements.txt       ← 必要なPythonパッケージ
├── documents/             ← 履歴書・ポートフォリオ等の添付ファイル
├── sites/
│   ├── crowdworks/        ← サイト別設定（config.json + SKILL.md）
│   ├── skillshift/
│   ├── hipro/
│   ├── furusatokengyo/
│   ├── otanomi/
│   ├── chiikizukan/
│   ├── lifull/
│   └── lotsful/
├── inbox/                 ← Coworkが結果を書き出す場所（jsonl形式）
├── processed/             ← 処理済みファイル
├── error/                 ← エラーファイル
└── logs/                  ← 実行ログ
```

## 対応サイトを追加したいとき

```
sites/{サイト名}/config.json  ← 検索条件
sites/{サイト名}/SKILL.md     ← フォームの埋め方
```
の2ファイルを作るだけ。`sync_sheets.py` は変更不要。
